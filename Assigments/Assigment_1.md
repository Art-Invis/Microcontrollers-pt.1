## Лабораторна робота №1

### Курс: *Мікроконтролери Ч.1*

### Тема: **Реалізація алгоритму засвічування 3 світлодіодів**

### 📌 Варіант 3

* **GPIO:** GPIO13
* **Тип обробки кнопки:** Click
* **Debounce:** External pull-up
* **Світлодіоди:** D4, D5, D6
* **Алгоритм:** №3 — *Збільшення швидкості блимання світлодіодів при кожному запуску*

---

### 🛠️ Мета роботи:

Ознайомлення з принципами роботи мікроконтролера **ESP8266 (Wemos D1 mini)**, написання прошивки мовою **C++ в Arduino IDE**, реалізація алгоритму керування трьома світлодіодами через кнопку та додатково — запуск через **вебінтерфейс**.

---

### 🔄 Опис виконання:

1. У середовищі симуляції або Arduino IDE створити схему на базі ESP8266 згідно з індивідуальним варіантом.
2. Реалізувати керування трьома LED з обробкою **кнопки (Click, external pull-up)**.
3. Запрограмувати алгоритм №3: кожен запуск прискорює частоту блимання світлодіодів.
4. Підключити до Wemos D1 mini:

   * Кнопку на GPIO13 через external pull-up
   * Світлодіоди до D4, D5, D6 (GPIO2, GPIO14, GPIO12)
5. Завантажити програму на мікроконтролер.
6. Перевірити коректність роботи: натискання кнопки має запускати алгоритм з новою, вищою швидкістю.
7. Створити просту **вебсторінку з кнопкою**, яка викликає запуск алгоритму через мережу (HTTP запит).
8. Продемонструвати результат викладачу.

---

### 🧠 Примітки:

* **D4** відповідає **вбудованому LED**, тому фізичне підключення до нього необов’язкове.
* Після завершення блимання повторний запуск можливий лише при **новому натисканні кнопки**.
* Вебсторінка може бути реалізована через простий HTML з кнопкою, яка надсилає запит на локальний IP ESP8266.

---

### 📎 Список використаного:

* ESP8266 Wemos D1 mini
* Arduino IDE
* HTML (статична сторінка)
* HTTP-запити
* Три світлодіоди та резистори
* Кнопка з зовнішнім підтягуванням (external pull-up)
